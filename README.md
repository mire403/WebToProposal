# WebToProposal
WebToProposal（网页信息自动写方案）是一款面向真实办公场景的资料整理与方案起草工具，用来把分散在多个网页中的信息，快速变成一份可直接修改使用的结构化方案初稿。用户只需提供若干 URL（新闻、官网、政策、行业报告等），系统会自动抓取并清洗网页正文，提取关键事实与观点，进行跨来源去重与融合归纳，最后按固定的办公文档结构输出：背景、现状分析、核心问题总结、可行方案建议。 它强调“基于已有网页信息的整理与结构化”，而不是凭空生成；产出内容偏客观、条理清晰，适合作为申报书、商业方案、产品方案、调研报告/可行性分析的初稿底稿，帮助用户把大量低价值的复制粘贴与初步梳理工作压缩到几分钟，但不替代人工审核与决策。
